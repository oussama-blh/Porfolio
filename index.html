01.

About

I'm a hybrid engineer — half builder, half breaker. My day-to-day moves between writing production code and auditing it for the things attackers actually exploit.

On the offensive side I work on web application pentesting, network reconnaissance, and OSINT — usually as part of authorized engagements or CTF research. On the build side I ship full-stack web apps with a focus on clean APIs, secure auth, and infrastructure that doesn't fall over at 3am.

I care about real security — not checklist theater — and I write code that's meant to be read. If you're shipping something where correctness and safety both matter, we'll get along.
